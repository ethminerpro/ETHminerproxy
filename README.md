# 如果你算力不準：
①第一檢查挖礦軟件配置及內核配置，是否設置超過多少分鐘沒有成功提交重啟內核

②查看你服務器的硬件配置及軟件帶寬，配置過低可能導致轉發性能不足，導致重發及超時

③檢查你服務器的網絡是否占用超過60%以上，是的話加帶寬
